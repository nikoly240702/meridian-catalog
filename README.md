# ТД «Меридиан» — сайт-каталог (демо)

Статический презентационный сайт-каталог поставщика. Чистые HTML/CSS/JS, без сборки и бэкенда.

## Страницы
- `index.html` — главная
- `catalog.html` — каталог раздела (товарная сетка)
- `product.html` — карточка товара
- `about.html` — о компании
- `contacts.html` — контакты + форма заявки (демо, без отправки)
- `assets/styles.css`, `assets/main.js` — общий стиль и скрипт

## Запуск локально
Открыть `index.html` через локальный сервер (из-за загрузки общего CSS):
```
cd tender-catalog
python3 -m http.server 8765
# открыть http://localhost:8765
```

## Заливка на хостинг
Сайт статический — подойдёт любой статический хостинг. Достаточно загрузить все файлы
(сохранив структуру с папкой `assets/`). Варианты под РФ: Timeweb, REG.RU, Beget, hosting от
Yandex Cloud (Object Storage + статический сайт), GitHub Pages.

## Что заменить перед боевым запуском
- **Фото** — сейчас демо-картинки по тегам (loremflickr, требуют интернет). Заменить на реальные
  фото товара/категорий в `assets/img/` и поправить пути `src` в HTML.
- **Реквизиты** — ООО, ИНН, ОГРН, банк, адрес, телефон, почта (сейчас нули-заглушки).
- **Товары** — демо-номенклатура в `<script>` внутри `catalog.html` и `product.html`.
- **Форма заявки** — сейчас не отправляет (демо). Для приёма заявок подключить обработчик
  (почтовый сервис / форму российского сервиса).
- Убрать пометки «ДЕМО-ВЕРСИЯ» в подвале и на странице контактов.
